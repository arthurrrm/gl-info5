# Projet Génie Logiciel

L’objectif de ce travail est de réaliser l’analyse, la conception et l’implémentation d’une application
correspondant aà un cahier des charges donné.

A faire :

- Demande création personnage (En cours Arthur)
- Accepter/refuser un personnage
- Consulter une biographie
  - Paragraphe secret
  - Modifier/Valider un épisode
  - Ajouter un épisode
- Gérer sa biographie
- Transférer un personnage
- Demander un changement de MJ
- Accepter/refuser la prise en charge d’un personnage
- Proposer une partie (en cours victor)
- Gérer les partcipants à une partie
- Finaliser une partie
- Supprimer une proposition de partie
